## 最简单的react ssr 配置
> 如果你想体验 ssr 渲染，最好理解请看这个分支
> 分支 simple-ssr
### 先执行npm run build
```
npm run build

npm start
```

> 打开

```
http://localhost:3006/
```

> 这个是最简单的服务端渲染

![](http://md.shudong.wang/stark-20180319110256807.png)
